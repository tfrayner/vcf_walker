==========
vcf_walker
==========

This package simply provides a common location for various classes
used to process VCFs in various ways (annotation, format conversion
and the like).

Installation
------------

As usual::

    python setup.py install

External Prerequisites
----------------------

   * PyVCF
   * BioPython
   * bx-python
   * bcbio-gff
